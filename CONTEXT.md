# 9rtm.dev Development

9rtm.dev の開発作業で使うリポジトリ固有の用語と、公開するコンテンツおよびその共有時の見え方を定義する。

## Language

**記事 OGP**:
個々の記事を共有したとき、その記事を識別できる画像。記事タイトル、絵文字、タグ、公開日、サイト名で構成し、概要文は含めない。
_Avoid_: OGP、サムネイル

**サイト OGP**:
記事以外のページを共有したとき、9rtm.dev を識別できる画像。
_Avoid_: デフォルト OGP、共通 OGP

**サイト署名**:
アバターとサイト名を組み合わせた、9rtm.dev の小さな識別要素。記事 OGP では主役にせず、画像の隅に配置する。
_Avoid_: ロゴ、著者アイコン

**記事カード**:
記事 OGP において、記事の情報を一つのまとまりとして載せる、背景から浮いた面。9rtm.dev の控えめな立体感を共有時の画像にも持ち込む。
_Avoid_: パネル、コンテナ

**タグチップ**:
記事カード上でタグを示す、小さく押し出された形の要素。
_Avoid_: タグバッジ、タグ欄

**本文書体**:
記事タイトルなど、内容を読み取ることが主目的の文字に使う書体。英数字には Mona Sans、日本語には M PLUS 1 を用いる。
_Avoid_: 通常フォント、タイトルフォント

**表示書体**:
タグや小さな見出しなど、9rtm.dev らしさを示す短い文字に使う Michroma。
_Avoid_: 装飾フォント、英字フォント

**面照明**:
カードやチップの下方を淡く明るくし、背景との小さな明暗差によって厚みを感じさせる表現。文字の装飾よりも面の立体感を担う。
_Avoid_: 背景グラデーション、光彩

**OGP 視覚トーン**:
記事 OGP とサイト OGP の情報配置を維持したまま、書体、色、面照明、境界線、影によって 9rtm.dev との一貫性を作る表現。情報の配置や優先順位は含めない。
_Avoid_: OGP レイアウト、OGP デザイン

**Worktree name**:
作業用ブランチと worktree の名前を導出するために利用者が指定する、Branch type を含まない名前。
_Avoid_: Branch name

**Branch type**:
作業の種類を表すブランチ名の接頭辞。`feat`、`fix`、`docs`、`refactor`、`chore` のいずれかで、既定値は `feat`。
_Avoid_: Prefix
